Top-level `layout { ... }`; `tab name="..." [focus=true]`; `pane name="..." cwd="..." command="..." { args "a" "b" }`.

Golden KDL sample (render of the fixture, cwd root `/work`):

## A. Ordering fix (guardian note #1) — DONE
`crates/engine/src/dashboard.rs`: the reader no longer uses `BTreeMap` (lexicographic). The
`projects:` mapping is now parsed via serde_yaml's order-preserving `serde_yaml::Mapping`
(IndexMap-backed) and iterated in document order; each value is deserialized per-entry via
`serde_yaml::from_value`. `group()` was rewritten from a `BTreeMap` bucket accumulator to an
insertion-ordered `Vec` so custom tabs and within-bucket repos keep declaration order
(KNOWN_TABS still take priority for known tabs). `BTreeMap` import removed.
- New unit test `reader_preserves_declaration_order_not_lexicographic` (keys deliberately
  anti-alphabetical: zeta, alpha, mike → preserved, NOT sorted).
- Existing order-dependent tests updated: `reader_parses_projects_map_and_url_forms` now asserts
  declaration order (meta_cli, meta_core, envctl, loop_lib, weave, repowire). Golden KDL unchanged
  (the fixture's meta-core repos coincide under both orderings).

## Deviations
- **`Engine::detached()` (engine + CLI change — NECESSARY for the plugin surface).** The
  `dashboard` verb is manifest-INDEPENDENT (it reads `.meta.yaml`, never the component registry),
  but `Engine::load_default()` hard-fails when no `manifest/` dir is present. The `meta dashboard`
  plugin shells `envctl dashboard` from the meta root (no manifest there), so it was failing with
  "manifest dir not found". Fix: added `Registry::empty()` + `Engine::detached()` (empty registry,
  real runner) in the engine, and the CLI uses it ONLY for `Cmd::Dashboard` (`main.rs`); every
  other verb still requires the real manifest. This keeps logic in the engine, preserves the
  fail-closed deploy (unchanged), and makes the plan's stated plugin behaviour actually work.
  Covered by new integration test `detached_engine_renders_dashboard_without_manifest`. This is a
  contained, additive change — not a plan defect — but flagged here for the guardian.
